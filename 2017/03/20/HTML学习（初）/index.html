body部分（基本标签）：
1.在网页上要展示出来的页面内容一定要放在<body>标签中。
2.<em> 表示强调，<strong> 表示更强烈的强调。并且在浏览器中<em> 默认用斜体表示，<strong> 用粗体表示。
3.要引用的文本不用加双引号，浏览器会对q标签自动添加双引号。
4.<blockquote>引用文本</blockquote>（长文本引用）
浏览器对<blockquote>标签的解析是缩进样式。
5.<br />标签分行显示文本。（相当于回车的作用）
<br />标签是一个空标签，没有HTML内容的标签就是空标签，空标签只需要写一个开始标签，这样的标签有<br />、<hr />和<img />。
6.<p>……</p> 是段落标签。（分段）
7.输入空格，必须写入&nbsp;。【直接输入空格的话是不会显示出来的】
8.<pre>语言代码段</pre> （常见用来展示计算机的源代码）
<pre> 标签的主要作用:预格式化的文本。被包围在 pre 元素中的文本通常会保留空格和换行符。

11.独立逻辑部分:使用<div>标签。
我们把一些标签放进<div>里，划分出一个独立的逻辑部分。为了使逻辑更加清晰，我们可以为这一个独立的逻辑部分设置一个名称，用id属性来为<div>提供唯一的名称，这个就像我们每个人都有一个身份证号，这个身份证号是唯一标识我们的身份的，也是必须唯一的。

12.表格:
<table>…</table>
<tbody>…</tbody>：当表格内容非常多时，表格会下载一点显示一点，但如果加上<tbody>标签后，这个表格就要等表格内容全部下载完才会显示。
<tr>…</tr>：表格的一行。
<td>…</td>：表格的一个单元格，一行中包含几对<td>…</td>，说明一行中就有几列。
<th>…</th>：表格的头部的一个单元格，表格表头。

14.链接到另一个页面：
<a href="目标网址" title="鼠标滑过显示的文本">链接显示的文本</a>。
15.插入图片：
<img src="图片地址" alt="下载失败时的替换文本" title = "提示文本">
src：标识图像的位置；
alt：指定图像的描述性文本，当图像不可见时（下载不成功时），可看到该属性指定的文本；
title：提供在图像可见时对图像的描述(鼠标滑过图片时显示的文本)
16.<span>标签为文字设置单独样式，没有语义。


body部分（表单标签）：
1.表单标签的作用是与用户进行交互。(表单是可以把浏览者输入的数据传送到服务器端，这样服务器端程序就可以处理表单传过来的数据)
2.所有表单控件（文本框、文本域、按钮、单选框、复选框等）都必须放在 <form></form> 标签之间，否则用户输入的信息提交不到服务器上。
3.文本域：<textarea rows="行数" cols="列数">文本</textarea>。
4.同一组的单选按钮，name 取值一定要一致，这样同一组的单选按钮才可以起到单选的作用。
单选/复选框语法格式：

type：当 type=”radio” 时，控件为单选框；
           当 type=”checkbox” 时，控件为复选框；
value：提交数据到服务器的值（后台程序PHP使用）；
name：为控件命名，以备后台程序 ASP、PHP 使用；
checked：当设置 checked=”checked” 时，该选项被默认选中。
5.form表单中的lable标签：
label标签不会向用户呈现任何特殊效果，它的作用是为鼠标用户改进了可用性。如果你在 label 标签内点击文本，就会触发此控件。就是说，当用户单击选中该label标签时，浏览器就会自动将焦点转到和标签相关的表单控件上（就自动选中和该label标签相关连的表单控件上）。
